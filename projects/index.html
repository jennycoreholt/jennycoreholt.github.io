COVID-19 Data Analysis

  • MySQL
  • Excel
  • Pandas

This project explores global COVID-19 data (2019–2021) using MySQL to perform data cleaning, transformation, and analysis. Pandas was used for part of the data cleaning (removing NAN). Excel and Numbers were used for splitting and exporting the data for analysis in MySQL Workbench. The goal was to extract meaningful insights, such as infection case counts, death rates, vaccination count and continent-level trends, which was later visualised in Tableau, as part of a separate data visualisation project.
